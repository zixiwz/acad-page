# Acad-Page

A static academic homepage built with plain HTML, CSS, and JavaScript.

## Structure

- `assets/data/site.js`: personal profile, navigation, education, publication, link, and footer data.
- `assets/images/`: avatars, icons, and image assets.
- `assets/files/`: downloadable files such as the CV.
- `framework/css/`: shared site styles.
- `framework/js/`: shared site scripts.
- `framework/pages/`: page-specific styles and renderers.
- `index.html`, `publications/`, `cv/`: static page entry points.

The publishable site lives under `docs/`. For GitHub Pages, set the source to `Deploy from a branch` and choose the `docs/` folder.

## Editing

For personal information and page content, edit `assets/data/site.js`.

For layout and visual style, edit files under `framework/`.
